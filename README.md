# Tractorio

## Workflow

### Git Locks

This Project uses Git Locks to prevent merge conflicts with following Unity Files:

| File Type        | File Extension |
|------------------|----------------|
| Prefab           | `.prefab`      |
| Scene            | `.unity`       |
| ScriptableObject | `.asset`       |

To set up the Git Locks Unity extension follow the [documentation on GitHub](https://github.com/TomDuchene/unity-git-locks?tab=readme-ov-file#initial-setup).
